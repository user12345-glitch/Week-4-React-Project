# Next.js Website Project

This is a simple Next.js website project built using React, TypeScript, and CSS.

## Features
- Semantic HTML structure
- Reusable React components
- Dark mode and light mode toggle
- Responsive design
- Form validation using React state


## Requirements

Before running the project, make sure you have the following installed:

- Node.js
- npm or yarn
- VS Code (recommended)

You can download Node.js from:

[Node.js Official Website](https://nodejs.org?utm_source=chatgpt.com)
